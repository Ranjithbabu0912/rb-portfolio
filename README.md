## Run Locally

Clone the project

```bash
  git clone https://github.com/Ranjithbabu0912/rb-portfolio.git
```

Go to the project directory

Install dependencies

```bash
  cd rb-portfolio
```

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
